Common Usage Scenarios

Scenario 1: Building Only

Toggles
✓ Building | ✗ Zone
  • See which building items are in
  • Don't need zone-level detail
  • Cleaner, more compact view

Scenario 2: Zone Only

Toggles
✗ Building | ✓ Zone
  • Focus on zone organization
  • All items in same building
  • Zone is primary grouping

Scenario 3: Both Hidden

Toggles

  • Location code only
  • Maximum space efficiency
  • Users know the context
