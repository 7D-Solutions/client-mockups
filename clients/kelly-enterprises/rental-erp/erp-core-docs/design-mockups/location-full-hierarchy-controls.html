Common Usage Scenarios

Scenario 1: Building + Zone Only

Toggles
✗ Facility | ✓ Building | ✓ Zone
  • Hide facility (all items in same facility)
  • Focus on building and zone organization
  • Most common use case for single-facility operations

Scenario 2: Building Only

Toggles
✗ Facility | ✓ Building | ✗ Zone
  • See which building items are in
  • Don't need zone-level detail
  • Cleanest view for building-level organization

Scenario 3: All Hierarchy Visible

Toggles
✓ Facility | ✓ Building | ✓ Zone
  • Full context for every item
  • Best for multi-facility operations
  • Complete traceability and organization

Scenario 4: Location Code Only

Toggles
✗ Facility | ✗ Building | ✗ Zone
  • Maximum space efficiency
  • Users already know the context
  • Quick scanning of location codes only
