  • All icons use FontAwesome 6.0 (fas/far classes)
  • Icons consistently appear before text with 8px gap
  • Loading states show a spinner and change text
  • Disabled buttons have 0.6 opacity and no hover effects
  • Badge counts appear inline for tabs and notifications
  • Color coding: Blue (primary actions), Green (save/success), Red (danger/delete), Teal (edit/info), Gray (cancel/secondary)
