## Know Your Rights

### You Have the Right To:
- [ ] Receive notice before meetings (10 days)
- [ ] Participate as an EQUAL team member
- [ ] Bring anyone you want to the meeting
- [ ] Ask questions until you understand
- [ ] Disagree and have it documented
- [ ] Request changes to the IEP
- [ ] Receive a copy of the IEP
- [ ] Request an Independent Educational Evaluation (IEE)
- [ ] NOT sign the IEP at the meeting (take it home)

### Remember:
- You do NOT have to sign the IEP at the meeting
- "I need time to review this" is always acceptable
- Request Prior Written Notice for any denial
- You can record the meeting (check your state's laws)

## Section 5: Questions to Ask

### About Progress:
- [ ] "How is my child progressing toward their goals?"
- [ ] "What data are you using to measure progress?"
- [ ] "How does my child compare to grade-level expectations?"
- [ ] "What's working and what isn't?"

### About Services:
- [ ] "Why is this amount of service recommended?"
- [ ] "What would happen if we increased/decreased this service?"
- [ ] "How will we know if this is enough?"
- [ ] "What's the plan to fade supports as my child becomes more independent?"

### About Placement:
- [ ] "Why is this the least restrictive environment for my child?"
- [ ] "What supports would my child need to be in a less restrictive setting?"
- [ ] "What supplementary aids and services were considered?"

### About the Plan:

- [ ] "What should I be doing at home to support these goals?"

## Section 7: If You Disagree

### Don't:
- [ ] Sign immediately if you disagree
- [ ] Get emotional or accusatory
- [ ] Storm out
- [ ] Threaten due process right away

### Do:
- [ ] Ask for clarification: "Help me understand..."
- [ ] Request data: "What evidence supports this decision?"
- [ ] Suggest alternatives: "What if we tried..."
- [ ] Request Prior Written Notice: "Please document in writing why this request is being denied."
- [ ] Ask to reconvene: "I'd like time to think about this. Can we continue this meeting next week?"

### Magic Phrases:
- "I'd like that documented in the IEP."
- "Please provide me with Prior Written Notice explaining this decision."
- "I'm not comfortable signing today. I'd like to take this home to review."
- "Can we schedule a follow-up meeting to discuss this further?"
- "I'd like to bring an advocate to our next meeting."

## Section 8: After the Meeting

### Within 24 Hours:
- [ ] Send thank-you email to team
- [ ] Summarize key decisions in writing ("Per our meeting, we agreed to...")
- [ ] Note any concerns not addressed
- [ ] Calendar follow-up dates

### Within 1 Week:
- [ ] Review the IEP document carefully
- [ ] Note any discrepancies from meeting discussion
- [ ] Request corrections if needed
- [ ] Sign and return (or request reconvene)

### Ongoing:
- [ ] Calendar progress report dates
- [ ] Start documenting for next meeting
- [ ] Implement home supports discussed
- [ ] Communicate regularly with teacher(s)

## Quick Reference: Your IEP Team Members

| Role | What They Do | Questions for Them |
|------|-------------|-------------------|
| **Special Ed Teacher** | Implements IEP, tracks goals, adapts instruction | "How is my child progressing?" "What's working?" |
| **General Ed Teacher** | Grade-level curriculum, classroom inclusion | "How does my child participate?" "What accommodations help most?" |
| **School Psychologist** | Testing, evaluations, behavior support | "What do the assessments show?" "What interventions might help?" |
| **Speech Therapist** | Communication skills, language | "How is communication progressing?" "What should I do at home?" |
| **Occupational Therapist** | Fine motor, sensory, daily living skills | "What sensory supports are working?" "Home activities?" |
| **Administrator** | Resources, placement, final decisions | "What options are available?" "What resources can we access?" |
